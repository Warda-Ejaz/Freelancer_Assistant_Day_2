# AI_Freelancer_Assistant_Day_2

A professional Streamlit application that helps freelancers instantly generate customized Proposals and Cover Letters using AI templates. It includes a live analytics Dashboard to track productivity.

### ✨ Key Features

- **📄 AI Proposal Generator**: Create tailored client proposals with dynamic inputs like client name, budget, and skills.
- **✉️ AI Cover Letter Generator**: Generate job-specific cover letters for any company and role.
- **📊 Live Dashboard**: Visualize total documents generated with interactive Plotly charts.
- **💳 AI Credits System**: Simulates an API credit system. 10 credits per proposal, 5 per cover letter.
- **📥 One-Click Download**: Download all generated documents as `.txt` files.
- **💾 Session State**: All data persists during the session without a database.

### 🛠️ Tech Stack

- **Frontend & Backend**: Streamlit
- **Data Visualization**: Plotly Express
- **Data Handling**: Pandas
- **Language**: Python 3.10+

### 🚀 Setup & Installation

1.  **Clone the repository**
    ```bash
    cd AI_FREELANCER_DAY_2
2.  *Install dependencies*
    pip install streamlit pandas plotly
3.  *Run the Application*

The app will open automatically at `http://localhost:8501`
